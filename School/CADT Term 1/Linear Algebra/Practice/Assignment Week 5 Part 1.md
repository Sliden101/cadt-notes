### 36. $\text{If }A_{\theta}=\begin{bmatrix}cos\theta&sin\theta\\-sin\theta&cos\theta\end{bmatrix}\text{Show that } A_{\theta}A_{\phi}=A_{\theta+\phi}$
$$\text{We have } A_{\theta}=\begin{bmatrix}cos\theta&sin\theta\\-sin\theta&cos\theta\end{bmatrix},A_{\phi}=\begin{bmatrix}cos\phi&sin\phi\\-sin\phi&cos\phi\end{bmatrix},A_{\theta+\phi}=\begin{bmatrix}cos(\theta+\phi)&sin(\theta+\phi)\\-sin(\theta+\phi)&cos(\theta+\phi)\end{bmatrix}$$
$$A_{\theta}A_{\phi}=A_{\theta}=\begin{bmatrix}cos\theta&sin\theta\\-sin\theta&cos\theta\end{bmatrix}\begin{bmatrix}cos\phi&sin\phi\\-sin\phi&cos\phi\end{bmatrix}=\begin{bmatrix}cos\theta cos\phi - sin\theta sin\phi&cos\theta sin\phi+sin\theta cos\phi\\-(sin\theta cos\phi +cos\theta sin\phi)&-sin\theta sin\phi +cos\theta cos\phi\end{bmatrix}$$
$$\text{But } cos\theta cos\phi - sin\theta sin\phi = cos(\theta+\phi)$$
$$\text{But } cos\theta sin\phi+sin\theta cos\phi = sin(\theta+\phi)$$
$$\text{But } sin\theta cos\phi +cos\theta sin\phi = sin(\theta+\phi)$$
$$\text{But } cos\theta cos\phi-sin\theta sin\phi=cos(\theta+\phi)$$
$$\text{Thus } A_{\theta}A_{\phi}=\begin{bmatrix}cos(\theta+\phi)&sin(\theta+\phi)\\-sin(\theta+\phi)&cos(\theta+\phi)\end{bmatrix}$$
$$\text{Therefore } A_{\theta}A_{\phi}=A_{\theta+\phi} $$
